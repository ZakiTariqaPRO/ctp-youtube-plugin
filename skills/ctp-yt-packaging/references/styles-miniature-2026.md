## Les 4 méta-règles (avant les styles)

1. **Un style a une durée de vie.** Il gagne parce qu'il est *différent* dans le feed. Quand toute
   la niche le copie (~6 mois), il s'éteint. → **Auditer SA niche AVANT de publier**, jamais un
   style « dans l'absolu ».
2. **⚠️ Pondérer la niche par sa MATURITÉ marketing.** Une niche peu mature (peu de chaînes
   packagent bien) n'est **PAS un benchmark de craft** — ses « winners » gagnent par sujet/chance,
   pas par miniature. Ne pas imiter la niche brute. → auditer contre une **whitelist de références
   curées** (les rares chaînes qui font de bonnes miniatures — fournie par le membre, vit dans SON
   projet `youtube/`) **+** emprunter aux niches adjacentes plus matures (méta-règle 3). Le membre
   sait souvent lister lui-même les 3-5 chaînes de référence : **les lui demander**.
3. **Le vrai jeu = niche-bending.** Emprunter un système visuel prouvé dans une niche **adjacente**
   plus mature et l'adapter — sans cloner la plus grosse chaîne (l'algo la montre à ta place, cf.
   `SIp7MeYbz8U`, `G1erIiTByPQ`).
4. **Tester dans le feed avant de publier.** Ce qui est beau en isolé peut disparaître entouré des
   autres. vidIQ a un preview feed (`dilqV_AGdEE`). Dans la boucle DFY : scorer + juger « je
   cliquerais en scrollant ? » (checklist `rBIeT9iLmnU` : focal point clair · contraste · curiosité
   ou bénéfice lisible en < 3 s).

## Le menu (11 styles) — psychologie + fit CTP

| # | Style | Ce qui déclenche le clic | Fit CTP / garde-fou |
|---|---|---|---|
| 1 | **Néo-minimalisme** | 1 sujet, ≥50 % espace négatif, 2 couleurs, fort contraste, fond blanc/mono. « Oasis » quand tout crie. | OK si c'est le pattern de SA niche (l'auditer). Colle à une charte épurée. |
| 2 | **Surround** (sujet centré + objets en cercle/grille organisée) | visage/objet au centre, éléments autour → l'œil explore, investissement +qq s. Pour comparatifs/« j'ai testé X ». | Rare pour un expert solo ; utile si la vidéo *liste/compare*. Chaos **contrôlé**, lisible en petit. |
| 3 | **Ranking / tier-list arc-en-ciel** | spectre de couleurs = complétude ; couleurs = système de rang, 3/5/7 items, chiffres. Engagement/controverse. | Le **chiffre + framework** est le levier fort. ⚠️ pas de tier-list « qui juge les gens » gratuite. |
| 4 | **Whiteboard / schéma** | dessin main = « éducatif haute valeur, un système qui marche », authentique (anti-veneer IA). Curiosité même si pas tout lisible. | Excellent pour l'autorité/expert. Montrer un vrai système. |
| 5 | **UI empruntée** (tweet, avis, badge vérifié, fiche) | crédibilité empruntée d'une interface reconnue ; « le web l'a déjà validé ». | ⚠️ **uniquement si le fond est vrai et vérifiable** — jamais une fausse validation (anti-clickbait CTP). |
| 6 | **Texte — glow-up cinématographique** | 3 mots (max 4), gros. **2026** : le texte **vit dans la scène** (partage le territoire du sujet, respecte l'éclairage/l'espace négatif) → « still Netflix » = signal qualité. Jaune/or = moderne, haut contraste, pop mobile. | **Le plus transposable à une charte premium noir/or.** Deux modes : (a) 3 mots gras dominants centrés, propres ; (b) cinématographique : composer l'image d'abord, poser les mots naturellement. **Ne pas répéter le titre** ; éviter la police « meme ». |
| 7 | **Visages déformés / double exposition** | pattern interrupt : « quelque chose cloche » → curiosity gap. Reflète le thème (crise d'identité, dualité, sujet lourd/psy). Lourd → à doser, doit être **intentionnel** (blend propre, éclairage fort), jamais gadget. |
| 8 | **Maximaliste / flex collection** | on empile ; **la collection EST la star** (le sujet passe 2nd). Crédibilité visuelle « j'ai tout ». Collector/hobby/tech. | Peu pertinent pour un expert-conseil. À réserver aux vidéos « panorama d'outils/ressources ». |

## ⭐ L'élément illustratif central (tendance forte — à ne pas oublier)

Quand la vidéo repose sur un **framework / une méthodologie / un outil**, la miniature ne doit pas
se contenter d'un **visage + texte** : elle porte un **élément illustratif central** qui **dépeint
le contenu**. C'est la psychologie « il a fait le tri / il a un système pour moi » (styles #4
whiteboard, #9 grille, #3 framework à chiffre) — et c'est très tendance.

- **Framework en N étapes** → une **image/diagramme en N étapes** (ex. « 6 principes » = 6 blocs).
- **Méthodologie** → **illustrer la méthodo** (schéma, boucle, pyramide, chemin).
- **Outil** → **illustrer l'outil** (rendu de l'écran/objet).
- **Concept** → un **objet symbolique** fort qui le résume.

Règle : un **vrai élément visuel qui habille la miniature**, centré, lisible en petit — pas un
détail décoratif ni un simple portrait. Il **coexiste** avec le visage (présentateur) et le texte
(2-4 mots), sans surcharge : 1 sujet + 1 illustration porteuse + texte court. Dans la boucle DFY,
ça doit apparaître **explicitement dans le `userQuery`** de `generate_thumbnail`.

## Checklist DA opérationnelle (craft — merci Yesir, membre du Collectif)

> Absorbée du duo de skills de **Yesir** (`recherche-miniatures` + `miniature-youtube`, Higgsfield/
> nano_banana_pro), filtrée CTP. Ce sont les règles de composition à appliquer À CHAQUE génération.

- **Un seul point focal** : visage **40-60 %** du cadre, sur une intersection règle des tiers.
- **Max 3 éléments** : visage + objet/illustration + texte. Au-delà = bruit illisible en mobile.
- **Direction du regard** : le regard du sujet **pointe vers le texte ou l'illustration** — il guide l'œil.
- **30-40 % d'espace négatif** autour du sujet.
- **⚠️ Zone morte bas-droite** : le **badge de durée YouTube** masque ~15 % du coin inférieur droit.
  N'y placer **aucun** élément clé (texte, visage, symbole).
- **Fond** : gradient sombre dramatique, sujet **détouré + rim light** (léger liseré) pour le décoller.
- **Expression** : pour du contenu sérieux/éducatif → **grave MAIS digne**, jamais **sinistre /
  menaçante / caricaturale** (sourcils détendus, regard engagé). Négatifs de prompt explicites :
  `not sinister, not menacing, no exaggerated frown`. (Piège : une photo source intense doit être
  ramenée vers « posé, digne ».)
- **Texte** : 2-4 mots, gras/majuscules, **ne répète pas le titre** (curiosity gap), zone basse hors
  coin bas-droite. Couleur d'accent sur 1 mot possible.
- **Mobile-first** : lisible à **120-320 px** de large. Vérifier mentalement en petit.
- **Toujours vérifier l'orthographe du texte rendu** (les modèles déforment parfois un mot) → régénérer.

## Ce qui vaut pour TOUS les styles (rappels corpus)

- **Packaging = titre + miniature en harmonie** (`-JFmcpwpQSs`) ; la miniature **ne répète pas** le
  titre, elle le complète. **Identité visuelle fixe** (palette + police récurrentes) = reconnaissance
  en millisecondes.
- **Cohésion shorts ↔ long** (`vi4PAtKsaCU`) : même **monde visuel** — si on tape le short puis le
  profil, les miniatures long-form doivent « venir de la même personne ». Un short punchy + un long
  « corporate » = handoff cassé.
- **Netteté / lisibilité petite taille** = non négociable (le score y pèse ~48 pts).
- **CTR bas + rétention haute = porte défaillante** (`rBIeT9iLmnU`) → **repackager** (re-titrer /
  re-miniaturiser), le dur est déjà fait.
- **Garde-fous CTP** : pas de chiffres d'argent, pas de figuration si la ligne du membre l'exclut,
  pas de clickbait creux, mots interdits bannis. La ligne vit dans les docs du membre (`voix/`).
