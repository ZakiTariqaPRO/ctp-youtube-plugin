# Collectif Tariqa PRO — Système YouTube (BETA)

> Le système qui aide un membre du **Collectif Tariqa PRO** à bâtir une **stratégie YouTube claire
> et méthodique**, puis à **produire son contenu au quotidien** : sujets, mots-clés, titres,
> miniatures, structure/script, et analyse. Un **outil vivant** — pas un exercice qu'on fait une fois.
>
> ⚠️ **BETA** en test. Retours bienvenus.

## ⛔ Prérequis (sine qua non) : le parcours CTP

Ce système **présuppose que tu as fait le parcours CTP** (persona → offre → positionnement → voix).
Sans ces fondations, aucun agent YouTube ne peut être pertinent — il produirait du générique.

👉 **Installe d'abord le parcours CTP :** https://github.com/ZakiTariqaPRO/ctp-plugin.git
*(persona, offre, positionnement/branding, voix, compliance, export, Coach Zaki)*

Chaque skill YouTube **vérifie** que `personas/`, `offres/`, `positionnement/`, `voix/` existent.
S'ils manquent → il t'arrête et te renvoie ici.

## Installation

1. **Le parcours CTP** (prérequis) — colle l'URL ci-dessus dans Claude Code et demande l'installation.
2. **Ce plugin YouTube** — colle l'URL de CE repo et demande l'installation.
3. Lance : **`lance ma stratégie YouTube`** (ou `travaille le packaging`, `structure ma vidéo`…).

## Ce que tu produis (livrables — comme le parcours CTP)

Le système écrit tes livrables dans ton projet (`youtube/`), réutilisables au quotidien :
- **Stratégie** : positionnement de chaîne, piliers, formats, funnel, cadence.
- **Mots-clés d'opportunité** (data vidIQ) + **plan éditorial** (titres mappés format × funnel).
- **Par vidéo** : idée → packaging (titres + **miniature réelle**) → hook → **structure/script** → CTA → SEO.
- **Feedback** post-upload (diagnostic métriques → itération).

## Le pipeline (ordre canonique)

```
① scan/stratège  ② Coach Zaki (challenge)  ③ packaging (+ miniature DONE-FOR-YOU)
④ hook  ⑤ structure vidéo (plan / mind map / script)  ⑥ CTA
— tournage + montage (toi) —
⑦ SEO  ⑧ feedback (48-72h après upload)
```

## Dépendances externes

- **MCP vidIQ** *(recommandé, PAYANT)* — data marché, keywords, analytics, génération/scoring de
  miniatures. **Le moteur est agnostique** : sans vidIQ, il travaille en **mode dégradé** et te livre
  les **prompts** pour générer la miniature dans **l'outil gratuit de ton choix**. Une **voie 100 %
  gratuite** est toujours proposée — aucun abonnement n'est imposé.
- *(Optionnel)* **MCP Windsor** *(payant)* — mesure du **CTR** réel (impressions), que vidIQ n'expose
  pas. *(Optionnel)* **Higgsfield** — génération de miniatures (alternative à vidIQ).

## Principe : moteur ≠ donnée

Le plugin = **la méthode** (distribuable telle quelle). **Ta data** (ICP, sales calls, métriques,
transcripts) vit dans **ton** projet, jamais dans le plugin. Plus tu nourris, plus c'est juste.
