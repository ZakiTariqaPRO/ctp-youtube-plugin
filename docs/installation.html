⚠️ Prérequis : un abonnement Claude payant La partie Code de Claude nécessite un abonnement Pro ou Max (un compte gratuit ne suffit pas). Assure-toi d'avoir ça avant de commencer.
Pourquoi Git ? « Git » est un petit outil gratuit qui permet à Claude d'aller chercher la boîte à outils du Collectif. Sans lui, l'installation automatique ne se fait pas. C'est à installer une seule fois.

Pour Mac 🍎

Mac — installation pas à pas

1

Installe l'application Claude

  • Va sur claude.ai/download et clique Download for Mac.
  • Ouvre le fichier téléchargé (.dmg) et glisse l'icône Claude dans le dossier Applications.
  • Ouvre Claude depuis tes Applications et connecte-toi.
2

Vérifie / installe Git

Beaucoup de Mac ont déjà Git. On vérifie d'abord, on installe seulement si besoin.

1) Vérifie :

  • Ouvre l'app Terminal (appuie sur ⌘ + Espace, tape « Terminal », Entrée — c'est la recherche Spotlight de Mac).
  • Copie-colle cette ligne, puis appuie sur Entrée :

git --version

  • Si un numéro s'affiche (ex. git version 2.39.x) → Git est déjà là, passe à l'étape 3. ✅
  • Si une fenêtre te propose d'installer les outils de développement en ligne de commande → clique Installer et attends la fin. Git sera alors prêt.

2) Si rien ne s'est passé, tape cette ligne dans le Terminal puis Entrée :

xcode-select --install

  • Une fenêtre s'ouvre → clique Installer, attends quelques minutes. (Si le message dit « already installed », c'est que Git est déjà là — tout va bien.)
Le Terminal, c'est juste pour vérifierTu t'en sers uniquement pour cette vérification de Git. Tu n'en auras plus besoin ensuite — la suite se fait dans l'app Claude.

Pour Windows 🪟

Windows — installation pas à pas

1

Installe Git pour Windows (à faire EN PREMIER)

  • Va sur git-scm.com/downloads/win et clique le bouton de téléchargement proposé par défaut (il choisit la bonne version tout seul).
  • Ouvre le fichier téléchargé. Si Windows affiche « Windows a protégé votre PC » → clique Informations complémentaires puis Exécuter quand même (c'est sûr).
  • Clique Next / Suivant à chaque écran sans rien changer (laisse tout par défaut), puis Install, puis Finish.
Ne saute pas cette étapeSans Git, la boîte à outils du Collectif ne pourra pas s'installer. Les écrans de l'installateur sont techniques : laisse simplement tout par défaut, c'est très bien.
2

Installe l'application Claude

  • Va sur claude.ai/download et clique Download for Windows.
  • Ouvre le fichier téléchargé et installe-le (« Suivant » jusqu'au bout). Même avertissement Windows possible → Exécuter quand même.
  • Lance Claude depuis le menu Démarrer et connecte-toi (abonnement Pro/Max).
Important : redémarre Claude après GitSi tu avais déjà ouvert Claude avant d'installer Git, ferme-le complètement et rouvre-le — sinon il ne verra pas Git.
