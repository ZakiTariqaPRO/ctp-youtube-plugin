# Guide membre — Système YouTube du Collectif Tariqa PRO (BETA)

> ⚠️ Brouillon de départ — la page explicative complète (objectif, installation, usage) est en cours.

## L'objectif
Bâtir une **stratégie YouTube claire et méthodique**, puis **produire ton contenu au quotidien**
(sujets, mots-clés, titres, miniatures, structure/script, analyse). Un **outil vivant**, pas un
exercice qu'on fait une fois. Tu en ressors avec un **playbook** réutilisable.

## ⛔ Prérequis : avoir fait le parcours CTP
Ce système présuppose le parcours CTP (persona → offre → positionnement → voix). Sans ces fondations,
le contenu serait générique.
👉 Installe d'abord : https://github.com/ZakiTariqaPRO/ctp-plugin.git

## Installer
1. Le **parcours CTP** (prérequis) — voir lien ci-dessus.
2. **Ce plugin YouTube** — ajoute ce repo comme marketplace / plugin dans Claude Code.
3. Lance : **`lance ma stratégie YouTube`**.

## Utiliser au quotidien
- **`lance ma stratégie YouTube`** / **`où j'en suis`** → la carte + l'étape suivante.
- **`on produit quoi aujourd'hui`** → un sujet/titre prêt, puis on enchaîne la production.
- Par vidéo : packaging (titre + miniature) → hook → structure/script → CTA → SEO → (après upload) feedback.
- **`exporte ma stratégie YouTube`** → ton playbook compilé.

## Dépendance
**vidIQ** (recommandé, payant) pour la data marché + la génération de miniatures. Le système marche
aussi **sans** (mode dégradé + prompts pour un outil gratuit). Aucun abonnement imposé.
